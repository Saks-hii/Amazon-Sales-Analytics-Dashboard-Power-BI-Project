# 📊 Amazon Sales Analytics Dashboard | Power BI

An interactive Power BI dashboard designed to analyze Amazon sales and operational performance. This project provides valuable business insights by visualizing sales trends, product performance, customer purchasing behavior, and fulfillment efficiency.

---

## 🚀 Project Overview

This dashboard helps businesses monitor key sales metrics and make data-driven decisions through interactive visualizations.

### 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing product categories
- Monitor order fulfillment and cancellation rates
- Compare sales across different states
- Generate actionable business insights

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

---

## 📂 Dataset Information

- **Dataset:** Amazon Sales Report
- **Domain:** E-Commerce
- **Records:** 120,000+ Orders

---

## 📊 Key Business Insights

- 💰 Total Sales: **₹79 Million**
- 📦 Total Orders: **120K**
- 🛒 Total Quantity Sold: **117K**
- 💵 Average Order Value: **₹653**
- 👕 **Set** category generated the highest sales.
- 🚚 Amazon Fulfilled orders significantly outperformed Merchant Fulfilled orders.
- ✅ Around **60%** of orders were successfully shipped.
- ❌ Approximately **17K** orders were cancelled.

---

## 💡 Business Recommendations

- Increase Average Order Value using product bundles and cross-selling.
- Promote premium products with attractive discount offers.
- Launch promotional campaigns during low-sales periods.
- Maintain sufficient inventory for high-performing categories.
- Expand Amazon Fulfilled inventory to improve delivery speed and customer satisfaction.
